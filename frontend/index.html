Product Mapping Tool

Hướng dẫn sử dụng

  1. Upload Files: Tải lên file nguồn (Source / df_a) và file đích (Target / df_b). Hỗ trợ định dạng .xlsx, .xls, .csv.
  2. Mapping Columns: Thêm các cặp cột để so sánh giữa 2 file. Mỗi cặp cần gán trọng số (weight), tổng các weight phải bằng 1.0. Thiết lập Threshold (mặc định 70) để phân loại Match/Unmatch.
  3. Export / Preview Columns: Chọn các cột muốn hiển thị trong kết quả preview và file export. Chỉ các cột được chọn ở đây cùng với cột score và predict sẽ xuất hiện trong kết quả.
  4. Run Mapping: Nhấn "Run Mapping" để chạy. Kết quả hiển thị trong bảng preview bên dưới, có thể sắp xếp theo cột score.
  5. Export: Nhấn "Export XLSX" để tải kết quả về file Excel.
